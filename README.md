# 🖐️ Hand Gesture Projects using Computer Vision

This repository contains a collection of interactive computer vision projects built using **Python, OpenCV, and MediaPipe**.
The goal of these projects is to explore real-time hand tracking and gesture-based interaction systems, with applications in **assistive technology and rehabilitation**.

---

## 📌 Projects Included

### 🧗 Rock Climbing Game

A virtual climbing game where the user interacts with on-screen holds using hand movement.

* Detects index finger position using MediaPipe
* Allows users to "climb" by reaching targets
* Tracks progress and movement

---

### 🍎 Apple Catch Game

A fun game where fruits fall from the screen and the user must catch only apples.

* Hand-controlled movement
* Score and life system
* Game-over condition after mistakes

---

### 🧩 Puzzle Game

A gesture-controlled drag-and-drop puzzle system.

* Pinch gesture to pick pieces
* Drag and place puzzle blocks
* Interactive visual feedback

---

### 🧱 Push Wall Game

A reflex-based game where the user pushes away incoming obstacles using hand motion.

* Real-time movement tracking
* Collision detection
* Interactive gameplay

---

### ☕ Cup Balance (Tilt Detection)

A prototype for an assistive system that detects hand tilt while holding an object.

* Calculates hand angle using landmarks
* Classifies:

  * Steady hold (Green)
  * Slight tilt (Yellow)
  * Spill risk (Red)
* Designed for rehabilitation and motor control analysis
